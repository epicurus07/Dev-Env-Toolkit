# dev-case-tools  
- docker를 이용한 소프트웨어 제작 인프라의 설치 방법 및 사용법
- 소규모 인프라이므로 대규모 프로젝트인 경우 Customizing이 필요  
- 도구는 Jenkins, Gitlab, SonarQube, netdata, Slack, Docker Registry, MySQL, Nginx을 사용  
- 샘플 프로젝트는 Java 언어에 Gradle 빌드 도구, Doxygen 사용
